download sublime:
http://www.sublimetext.com/3

instale o packege control, para isso entre no seguinte site e siga as intrucoes:
https://sublime.wbond.net/installation

siga seguintes instrucoes para os seguitnes pacotes:
-Anaconda
-Djaneiro
-Emmet

1-aperte ctrl+shift+P para abrir os Commandos
2-digite install, deverá aparecer uma opcao 'Package Control: Install Packege'
3-aperte enter
4-digite o nome do pacote que deseja instalar
5-Selecione o pacote e aperte enter

Entre em Preferences>Settings-User (muitas dessas configurações são boas pra qualquer linguagem, mas eh possivel ter configurações especificas para linguagens diferentes):
  "auto_complete_commit_on_tab":true
  "tab_size": 4,
  "translate_tabs_to_spaces": true,
  "trim_automatic_white_space": true,
  "trim_trailing_white_space_on_save": true
  "ensure_newline_at_eof_on_save": true,
  "folder_exclude_patterns":
  [
    ".svn",
    ".git"
  ],
